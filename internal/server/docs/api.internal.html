Create dead man switch

Request Body schema: application/json
required
checkInInterval
required
string^[0-9]+[smh]$

Timer countdown until a switch is triggered

deleteAfterTriggered
boolean

Whether to delete the switch after triggering

encrypted
boolean

Where or not to encrypt switch data. Data will no longer be readable via the API

message
required
string non-empty
notifiers
required
Array of strings

List of notification channels powered by shoutrrr

object

Optional PWA push subscription for background alerts

reminderThreshold
string^[0-9]+[smh]$

How long before expiration to send a push notification
